# Plant Disease Detection using CNN

Projet académique : Détection de maladies des plantes à partir d'images de feuilles.

## Structure du projet

- data/ : dataset raw et processed
- notebooks/ : notebooks d'exploration et modèles
- src/ : scripts python
- models/ : modèles entraînés
- reports/ : rapport académique
- slides/ : présentation et figures
- requirements.txt : librairies nécessaires

## Instructions pour commencer

1. Créer et activer l'environnement conda
2. Installer les librairies : `pip install -r requirements.txt`
3. Lancer les notebooks ou les scripts dans `src/`
